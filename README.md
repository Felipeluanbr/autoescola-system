# AUTOESCOLA-SYSTEM – Interface Administrativa Frontend

## O que é?
 Interface Frontend que simula um sistema real de autoescola.

## Por que existe
 Este projeto foi desenvolvido para aplicar conceitos de arquitetura, autenticação, banco de dados e boas práticas no desenvolvimento web. 

## O que você quis provar tecnicamente.
  - Responsividade
  - Organização de arquitetura
  - Separação por modulos
  - Componentizaçãos
  - Uso de Tailwind
  - Estrutura escalável


## Tecnologias
 - HTML
 - CSS
 - Tailwind
 - JavaScript
